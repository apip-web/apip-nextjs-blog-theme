# Mengubah Font dan Tema pada Blog Next.js Theme Ini

Halo! Kali ini saya mau berbagi pengalaman mengubah font dan tema warna pada template blog Next.js ini (https://github.com/apip-web/apip-nextjs-blog-theme).

Secara default, blog ini menggunakan **font sistem (sans-serif)** dan tema warna **default** (biru ungu). Tapi ternyata kita bisa mengubahnya dengan mudah, bahkan pakai font premium seperti **Geist** yang sama persis dengan yang dipakai di blog SvelteKit saya (https://apip-web.vercel.app).

### 1. File-file Penting yang Mengatur Font & Tema

Ada beberapa file kunci yang mengatur tampilan font dan warna:

- **`themes.js`**  
  Tempat definisi semua tema warna (`COLOR_THEMES`) dan font yang didukung (`FONT_THEMES`).

- **`utils/theme-utils.js`**  
  File yang menghasilkan variabel CSS `--theme-*` berdasarkan konfigurasi di `themes.js` dan environment variables.

- **`styles/globals.css`**  
  File yang menerapkan variabel CSS tersebut ke elemen HTML (heading dan body).

### 2. Mengubah Font Menjadi Geist

Saya ingin fontnya sama persis dengan blog SvelteKit: **Geist** dari Google Fonts.

Langkah-langkahnya:

#### a. Tambahkan Geist ke `themes.js`
```js
exports.FONT_THEMES = {
  // ... tema lain
  Geist: `'Geist', ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif`,
};
```

#### b. Impor Geist di `pages/_app.js`
```jsx
function MyApp({ Component, pageProps }) {
  return (
    <>
      <link
        rel="stylesheet"
        href="https://fonts.googleapis.com/css2?family=Geist:wght@100..900&display=swap"
      />
      <span className="theme-bejamas" />
      <Component {...pageProps} />
    </>
  );
}
```

#### c. Hard-code Geist di `utils/theme-utils.js`
```js
cssVars['--theme-headings'] = `'Geist', Courier, sans-serif`;
cssVars['--theme-body'] = `'Geist', Courier, sans-serif`;
```

> Tips: Saya sengaja pakai `Courier` sebagai fallback karena tampilannya mencolok banget. Kalau font Geist gagal dimuat, langsung kelihatan (untuk debugging).

Setelah ini, font seluruh blog jadi Geist â€” heading tebal, body clean, persis seperti blog SvelteKit.

### 3. Mengubah Tema Warna
